User Management

На портале администратора каждый пользователь уникально идентифицируется с именем пользователя, паролем и профилем. Вместе с другими настройками профиль определяет, какие задачи пользователь может выполнять, какие данные пользователь может видеть, и что они могут с ним делать. Раздел «Управление пользователями» позволяет администратору создавать и управлять пользователями, группами мерчанта и задавать роли. Вы также можете просмотреть любой сеанс пользователя из панели администратора. (Дополнительную информацию о отслеживании пользовательских сеансов можно найти тут ).


Submenu

User

Пользователь - это лицо, у которого есть учетная запись в системе портала администратора. Каждый пользователь имеет свой собственный уникальный идентификатор, который установлен на момент создания учетной записи, и включает имя пользователя, имя, фамилию, адрес электронной почты и их роли.

Role

Роли используются для управления разрешениями для функций, которые могут выполняться на портале. Ролям может быть разрешено выполнять различные функции во всех шлюзах, конкретном шлюзе, с торговцем или с учетной записью. Роли - это просто набор разрешений, которые определяют разрешенные функции, такие как Администратор мерчанта и т.д.

Merchant Group

Merchant Group - это набор мерчантов, которые каким-то образом связаны. В разделе «Merchant Group» вы можете создавать группы, которые позволяют пользователям просматривать отчет статистики группы без необходимости отдельно выбирать каждого торговца.
