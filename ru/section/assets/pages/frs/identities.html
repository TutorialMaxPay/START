Section: Home FRS Identities Create

Назначение:

Создание identities для продавца с настройками даты регистрации и даты окончательного расчета. Для каждой Identity можно добавить план, уникальную плату или копирование между торговцами комиссий.


Для создания identities Следуйте инструкции


Rule and description


Rule Description
Merchant Название мерчанта
Enroll Date Дата регистрации
Termination Date Срок действия
Final Settlement Date Дата окончания
Billing Address Адресс регистрации

Информация:

Дальше вы можете настроить Identities, добавить комисии, добавить Plan, кипировать комиссии или изменять их .
