Section: Home FRS Balance Records Create

Назначение:

Данный раздел позволяет создать балансовые записи вручную.

Для создания балансовой записи заполните поля


Field Description Example
Identity * Прежде настроенные правила для комиссий Test_merchant
Merchant * Имя Мерчанта в админ панели Test_merchant
Merchant Account Аккаунт мерчанта в админ панели test_account
Bank Название банка Be2bill
MID Название мида в админ панели test_mid
Wallet * Кошельки мерчанта Be2bill
Billing Date * Дата балансовой записи 2018/04/17 00:00
Amount * Сумма записи 10
Currency * Валюта записи USD
Origin * Ручная запись или корректировка Manual
Fee Rule Комиссионные правила в identity Rule_for_sale
Type * Debit или Credit Debit
Test Mode * Yes|No Yes
Paid Yes|No No
Penalty Yes|No DescTestMerchant
Comments * Ваш комментарий ***
Documents Прикрепить invoice Live gate

Section: Home FRS Balance Records Calculate

Назначение:

Данная секция позволяет пересчитать комиссии, если они были изменены либо добавлены новые.

Заполните поля для пересчета комиссий


Field Description Example
Identity * Прежде настроенные комиссионные правила Test_merchant
Fee Комиссии в identity test_rule_payout
Start Date * Дата начала 2018/04/17 00:00
Finish Date * Дата конца 2018/04/27 00:00
Strategy * Текущие или исторические Historical
Entity * Тип комиссии Recurring fee
Apply Balance Delta * Yes|No Yes

Заметка | Всегда заполняйте поля отмеченные *

После заполнения нажмите на кнопку



Section: Home FRS Balance records List

Назначение:

Данный раздел позволяет найти балансовые записи. Также, если комиссии были изменены, функция "Recalculate" позволяет отменить и пересчитать балансовые записи

Настройте фильтры для поиска


Field Description Sample
Identity * Прежде настроенные комиссионные правила Test_merchant
Without Invoice Подключать "Invoice" Yes|No
Invoice Выберите из списка All
Wallet Выберите кошелек Bank_fee
Condition Дополнительные кондиции Billing Date

После настройки фильтров нажмите на кнопку Или необходимое действие
