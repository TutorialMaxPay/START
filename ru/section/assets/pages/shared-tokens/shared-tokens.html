Section: Home Shared Tokens Create

Назначение:

Общие токены позволяют менять токены между учетными записями других мерчантов или аккаунтов мерчанта.


Для создания распределения токенов следуйте инструкциям


Шаг 1 : Выбирете мерчанта для распределения




Следует выбрать аккаунт или все аккаунты мерчанта


Для выбора всех аакаунтов поставьте галочку


Для выбора аккаунта нажмите


И выбирете из списка




Для удаления распределения нажмите



Шаг 1: Скриншот :



После выбора всех настроек нажмите на кнопку


Шаг 2 : Выбирете мерчанта которому будут распределятся токены




На шаге 2 сделайте то же, что и на шаге 1, но только с учетными записями которым предназначено распределение.


Шаг 2: Скриншот :



После выбора всех настроек нажмите на кнопку



Section: Home Shared Tokens List

Назначение:

В этом разделе можно найти все настройки общех токенов между учетными записями продавца.


Для поиска выберите правила


Rule Description
Gate Поиск по шлюзу
Distributor Merchant Поиск по мерчанту распределения
Distributor Account Поиск по аккаунту распределения
Consumer Merchant Поиск по мерчанту для распределения
Consumer Account Поиск по аккаунту для распределения

Пометка | Вы можете выбрать несколько правил

После выбора правил нажмите на кнопку


Для создания новых общих токенов нажмите на кнопку



Пример поиска

Test gate ABC ABC_account TEST qatest 2016-09-23 16:14:13
Live gate Merchant_live Live_account Merchant_live2 Live_account2 2016-09-28 16:14:13


Детали:

Update - обновление настроек.

Delete - удаление элемента.
