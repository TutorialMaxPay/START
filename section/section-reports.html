Section: Home Reports


Reports

Many types of reports can be used in the Section Reports. By using filters to analyze your data, you can draw out more information that can help you make the right decisions to improve your service.


Submenu

Transactions

The Transaction Report shows you a list of your transactions within a given date range, as well as the source and type of the transaction. In the transaction report, you are able to include selected fields, view the details of the transaction, make a refund or void a transaction.


Get more details


Chargebacks

A Chargeback is a return of funds to a consumer that is initiated by the issuing bank of the instrument used by the consumer. In the chargeback report, you are able to add the most important fields to analyze the case.


Get more details


Approve Ratio

The Approval Ratio report provides the ratio of successful transactions to declined, error, and filtered transactions. Real-time graphs and charts allow you see and understand the statistics easier.


Get more details


Chargeback Ratio

The Chargeback Ratio report shows the ratio of chargeback transactions to successful transactions. VISA and Mastercard calculate the chargeback ratio for each MID each month.


Get more details


Fraud Ratio

Credit card fraud is a broad term for the theft of and fraudulant use of a payment card, such as a credit card or debit card. The Fraud Ratio report provides the ratio of fraudulent transactions to valid ones.


Get more details


Summary

The Summary Report covers sales, refunds, chargebacks, declines, errors and filtered transactions. The report shows the number of transactions by scope and amount. Select the filters to get more detailed information by merchant, bank MID etc.


Get more details


Transactions Cube

The Transaction Cube Report is a powerful tool for creating a extensive, particular report. Simply select the triggers and filters that you want to use for the report.


Get more details

Scheduled

Scheduled reports allow you to get data statistics sent to your email. Just select the kind of the report and it will come at the specified time.

Custom reports can be made by selecting the settings which you need and saving it. Next time, you just need to open the saved report.
