Section: Home Mms Currency converter Create

For converting currency fill up the fields


Field Description Example
Merchant Account * select a merchant account where situated the MID ABC_account
Mid * select the MID for converting the currency ABC_mid
Currency From select the currency for change from USD
Currency To select the currency for change to (default value is EUR) EUR
Conversion Fee Fee of the currency exchange. Formula: amount*(exRate + exRate/(100% * X%)) 10%

Note | Always fill up the fields with *
