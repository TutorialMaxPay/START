Section: Home Tools Chargeback Create

For creating a chargeback manually field up the fields below


Field Description Example
Reference reference of the transaction SLF000000022224234
Chargeback Id ID of the chargeback CB0000021212342413
Chargeback Amount amount of the chargeback transaction 20
Chargeback Currency currency of the chargeback transaction USD
Chargeback Type type of the chargeback RETRIEVAL_REQUEST FIRST_CHARGEBACK SECOND_THIRD_CHARGEBACK REPRESENTMENT CHARGEBACK_REVERSAL LOST_REPRESENTMENT
Chargeback Status status of the chargeback transaction OPENED REPRESENTED REVERSED CLOSED
Chargeback Date date of the chargeback transaction, In UTC Timezone 18.04.2016 | 13:43:43
Chargeback Update Date update date of the chargeback transaction, In UTC Timezone 11.05.2016 | 11:21:48
Chargeback Reason reason of the chargeback transaction optional
Chargeback Reason Message reason message of the chargeback transaction optional
Parent Chargeback Id parent ID of the chargeback transaction optional
